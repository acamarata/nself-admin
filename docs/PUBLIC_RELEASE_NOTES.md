## Installation

### Docker (Recommended)

```bash
# Latest stable release
docker pull acamarata/nself-admin:latest

# Specific version
docker pull acamarata/nself-admin:0.9.0

# Run with nself CLI
nself admin
```

### From Source

```bash
# Clone repository
git clone https://github.com/acamarata/nself-admin.git
cd nself-admin

# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build
pnpm start
```
